# DEFWallet-for-iOS
DEFWallet for iOS client

##How to install
step1:
```Bash
cd /yourpath
git clone https://github.com/defchain/DEFWallet-for-iOS.git
``` 

step2:
```Bash
chmod +x /yourpath
./build.sh
``` 

step3:
 run DEFWallet.xcodeproj
